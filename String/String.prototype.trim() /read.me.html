Конспект Метод trim() удаляет пробелы в начале и конце строки. Возвращает значение строки без пробелов в начале и конце, при этом значение самой строки остаётся неизменным. Синтаксис: string.trim(). let str = ' Sun ! '; console.log(str.trim()); // "Sun !" console.log(str); // " Sun ! " Подробнее читайте здесь: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/Trim
